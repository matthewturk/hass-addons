# hass-luanti

A Home Assistant add-on repository that hosts a [Luanti](https://www.luanti.org/)
(formerly Minetest) game server.

[![Open your Home Assistant instance and add this repository.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fmatthewturk%2Fhass-luanti)

## Add-ons

This repository contains the following add-ons

### [Luanti Server](./luanti)

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

_Host a Luanti (formerly Minetest) voxel game server directly from Home Assistant.
All server options are configurable from the HA UI, and mods/games can be added
via the `/share/luanti/` directory._
