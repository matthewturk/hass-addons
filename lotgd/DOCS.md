# Documentation: Legend of the Green Dragon

## Configuration

The add-on can be configured using the following options:

- `db_host`: The hostname of the database server. Use `core-mariadb` if using the official MariaDB add-on.
- `db_name`: The name of the database.
- `db_user`: The database user.
- `db_password`: The database password.
- `db_use_data_cache`: Whether to use a data cache.

## Getting Started

1. Install the official MariaDB add-on if you haven't already.
2. Create a database for LotGD in MariaDB.
3. Configure this add-on with the database details.
4. Start the add-on.
5. Access the game at `http://<your-hass-ip>:80`.
6. Follow the `installer.php` if it's the first time.
