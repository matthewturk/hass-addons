# Home Assistant Add-on: RomM

_ROM Manager - A beautiful, powerful, self-hosted ROM management solution._

## Installation

To install this add-on locally:

1. Copy this `romm` folder to the `/addons` directory on your Home Assistant instance.
2. Go to **Settings** > **Add-ons** > **Add-on Store**.
3. Click the menu (three dots) and select **Check for updates**.
4. Install **RomM** from the **Local add-ons** section.

For more details, see the [documentation](./DOCS.md).
